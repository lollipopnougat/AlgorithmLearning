# 题目

## 第一个只出现一次的字符

### 来源:

[力扣-第一个只出现一次的字符](https://leetcode-cn.com/problems/di-yi-ge-zhi-chu-xian-yi-ci-de-zi-fu-lcof/)

### 题目内容

在字符串 s 中找出第一个只出现一次的字符。如果没有，返回一个单空格。 s 只包含小写字母。

**示例:**

    
    
    s = "abaccdeff"
    返回 "b"
    
    s = "" 
    返回 " "
    



**限制：**

`0 <= s 的长度 <= 50000`
