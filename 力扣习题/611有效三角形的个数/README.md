# 题目

## 有效三角形的个数

### 来源:

[力扣-有效三角形的个数](https://leetcode-cn.com/problems/valid-triangle-number/)

### 题目内容

给定一个包含非负整数的数组，你的任务是统计其中可以组成三角形三条边的三元组个数。

### 示例 1:

```plaintext
输入: [2,2,3,4]
输出: 3
```

解释:

```plaintext
有效的组合是: 
2,3,4 (使用第一个 2)
2,3,4 (使用第二个 2)
2,2,3
```

### 注意:

- 数组长度不超过 `1000`。
- 数组里整数的范围为 `[0, 1000]`。
