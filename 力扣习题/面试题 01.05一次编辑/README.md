# 题目

## 一次编辑

### 来源:

[力扣-一次编辑](https://leetcode.cn/problems/one-away-lcci/)

### 题目内容

字符串有三种编辑操作:插入一个字符、删除一个字符或者替换一个字符。 给定两个字符串，编写一个函数判定它们是否只需要一次(或者零次)编辑。



**示例  1:**

    
    
    **输入:** 
    first = "pale"
    second = "ple"
    **输出:** True



**示例  2:**

    
    
    **输入:** 
    first = "pales"
    second = "pal"
    **输出:** False
