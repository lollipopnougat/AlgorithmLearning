# 题目

## 不同的二叉搜索树 II

### 来源:

[力扣-不同的二叉搜索树 II](https://leetcode-cn.com/problems/unique-binary-search-trees-ii/)

### 题目内容

给定一个整数 `n`，生成所有由 `1 ... n` 为节点所组成的 `二叉搜索树` 。

### 示例：

> 输入：3
> 输出：

``` py3
[
  [1,None,3,2],
  [3,2,None,1],
  [3,1,None,None,2],
  [2,1,3],
  [1,None,2,None,3]
]
```

### 解释：

以上的输出对应以下 5 种不同结构的二叉搜索树：

```plaintext
   1         3     3      2      1
    \       /     /      / \      \
     3     2     1      1   3      2
    /     /       \                 \
   2     1         2                 3
```

提示：

`0 <= n <= 8`
