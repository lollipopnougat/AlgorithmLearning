# 题目

## 不同的二叉搜索树

### 来源:

[力扣-不同的二叉搜索树](https://leetcode.cn/problems/unique-binary-search-trees/)

### 题目内容

给你一个整数 `n` ，求恰由 `n` 个节点组成且节点值从 `1` 到 `n` 互不相同的 **二叉搜索树** 有多少种？返回满足题意的二叉搜索树的种数。

**示例 1：**

![](./uniquebstn3.jpg)

    
    
    **输入：** n = 3
    **输出：** 5
    

**示例 2：**

    
    
    **输入：** n = 1
    **输出：** 1
    

**提示：**

  * `1 <= n <= 19`
